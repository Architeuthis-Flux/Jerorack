# Jerorack

This is me screwing around with designing a eurorack front panel.

The idea is having a grid of 3D magnetometers so you can plug knobs or springy door stoppers or whatever into any jack or use a magnetic probe to draw wires which would show up on an insanely dense flowy LED matrix on the front.

This would plug into a hypothetical future [Jumperless Barnacle](https://github.com/Architeuthis-Flux/Barnacle_Board) on the back to do the routing.
